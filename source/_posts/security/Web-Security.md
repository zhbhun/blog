# 安全原则
- 不要信任用户的输入信息
- 验证非信任源的输入信息

# 攻击方式
- SQL 注入攻击
- XSS：跨站脚本攻击
- CSRF：跨站伪造请求攻击
- URL地址攻击
- 数据越权访问
- 文件上传漏洞攻击
- DDOS：分布式拒绝服务攻击
- 密码暴力破解
- 错误代码暴露
- 残留测试文件
