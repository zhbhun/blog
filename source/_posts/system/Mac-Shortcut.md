# 切换
- command + tab：在应用程序间切换
- command + ~：在应用程序中的窗口间切换

# 通用
- command + H：隐藏窗口
- command + M：最小化窗口
- commang + N：新建窗口
- commang + O：打开文件
- command + S：保存文件
- command + shift + S：另存为
- command + P：打印
- command + W：关闭
- command + Q：退出
