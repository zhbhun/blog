# 设备信息
| 屏幕密度 | DPI | 比率 |
| 低/ldpi | 120 | 0.75 |
| 中/mdpi | 160 | 1 |
| 高/hdpi | 240 | 1.5 |
| 超高/xdpi  | 320 | 2 |

像素 = DP * (DPI / 160)

- 320*480
- 480*800
- 720*1280
- 1080*1920
